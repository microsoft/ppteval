# PPTArena Assets

This directory contains placeholder files for images and other assets.

## Directory Structure

- `favicon.ico` - Site favicon
- `og-image.png` - Open Graph image for social media sharing
- `logo.png` - Site logo
- `hero-bg.jpg` - Hero section background image

## Adding Real Assets

Replace these placeholder files with your actual project assets:

1. **Favicon**: 32x32 pixel icon for browser tabs
2. **Open Graph Image**: 1200x630 pixel image for social media
3. **Logo**: High-resolution transparent PNG
4. **Background Images**: Optimized images for web use

## Image Optimization

For best performance:
- Use WebP format when possible
- Compress images appropriately
- Provide multiple sizes for responsive images
- Use lazy loading for images below the fold
