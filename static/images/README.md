# Static Images

This directory mirrors the structure used by the OSWorld reference site.

## Sample Images Needed

- `task_demonstration.png` - Screenshot showing example PPT tasks
- `env.png` - System architecture diagram
- `statistics.png` - Data overview charts
- `pie_chart.png` - Task distribution visualization

## Usage

These images are referenced in the main content and should be:
- High quality (at least 1000px wide)
- Optimized for web (compressed appropriately)
- Accessible (with proper alt text)

## Creating Images

For consistency with academic presentation:
- Use clean, professional styling
- Include clear labels and legends
- Maintain readable font sizes
- Use consistent color schemes
